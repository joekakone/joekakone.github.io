Projets

Databases Analytics Tools

DB Analytics Tools est un micro-framework open source qui aide les analystes de données à travailler avec des bases de données relationnelles (entrepôts de données).

  • Exécuter des requêtes SQL (DDL/DCL/DML)
  • Exécuter les procédures et fonctions SQL (ETL)
  • Importation et Exportation de données
  • Compatible avec Postgres, Greenplum et SQL Server

Lien du projet: https://joekakone.github.io/db-analytics-tools

Algo Jungle

Algo Jungle est une plateforme d'apprentissage en ligne dédiée à la diffusion des connaissances sur Python et la Data. Voici les sujets abordés :

  • Algorithmes & Structures de données
  • Programmation (C & Python)
  • Business Intelligence & Data Analytics
  • Data Science & Machine Learning

Lien du projet: https://joekakone.github.io/learning

Jobs Tracker

Jobs Tracker est un système d'alerte emploi qui permet aux utilisateurs de configurer des notifications concernant des offres d'emploi spécifiques au Togo.

  • Extraire les offres d'emploi de Emploi.tg et Emploitogo.info
  • Associer les offres d'emploi aux mots clés
  • Envoyer un mail contenant la liste des offres correspondantes

Outils: Python, Requests, BeautifulSoup, Bokeh, AddThis

Lien du projet: https://joekakone.github.io/jobs-tracker

Deputes.fr

Deputes.fr est un dashbord interactif qui vous permet de visulaiser les données de l'Assemnlée Nationale Française. Il est développé avec le package Bokeh.

  • KPIs: Nombre de députés, Nombre de groupes, Âge moyen, Expérience moyenne
  • Graphiques : Donut Chart, Barplot (H & V)
  • Tableau : Liste des députés

Outils: Python, Pandas, Bokeh

Lien du projet: https://github.com/joekakone/deputes-fr
